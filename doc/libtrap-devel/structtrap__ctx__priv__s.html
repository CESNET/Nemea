Detailed Description

Libtrap context structure.

It contains the whole context of one instance of libtrap. The context contains arrays of communication interfaces (IFC), buffers, locks/mutexes.

int32_t trap_ctx_priv_s::readers_count

Number of reader threads that will read messages during multiread (set by collector thread). Every client decreases this value and the last reader (readers_count equals 0 after decrementation) wakes the collector thread.
